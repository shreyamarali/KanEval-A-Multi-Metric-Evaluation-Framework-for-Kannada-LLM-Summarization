# KanEval: Multi-Metric Evaluation Framework for Kannada LLM Summarization

KanEval is a Streamlit-based NLP evaluation and benchmarking framework designed to compare the performance of multiple Large Language Models (LLMs) for Kannada text summarization tasks.

The system evaluates summaries generated by different LLMs using semantic, lexical, structural, and information coverage metrics, and provides comparative visual analytics to identify the best-performing model.

---

## Features

- Comparative benchmarking of multiple LLMs
- Semantic similarity evaluation using multilingual sentence embeddings
- Lexical diversity analysis
- Average sentence length analysis
- Information coverage analysis
- Final weighted model ranking
- Interactive visualization dashboard
- CSV export support
- Kannada language support
- Streamlit-based user interface

---

## Technologies Used

- Python
- Streamlit
- Sentence Transformers
- Pandas
- Regular Expressions (re)
- Multilingual MiniLM Embeddings

## Project Architecture

```text
User Input Kannada Text
            ↓
Multiple LLM Summaries
            ↓
Evaluation Engine
    ├── Semantic Similarity
    ├── Lexical Diversity
    ├── Sentence Structure Analysis
    └── Information Coverage Analysis
            ↓
Comparative Metrics Dashboard
            ↓
Model Ranking & Visualization
```

---

## Evaluation Metrics

### Semantic Similarity
Measures contextual similarity between original text and generated summaries using multilingual sentence embeddings.

### Lexical Diversity
Measures vocabulary richness and variation in generated summaries.

### Average Sentence Length
Analyzes structural complexity and readability.

### Information Coverage Analysis
Identifies missing and additional information in summaries compared to the original text.

### Final Weighted Score
Generates an overall ranking score using weighted evaluation metrics.

## Installation

Clone the repository

Move into the project folder:

```bash
cd KanEval-A-Multi-Metric-Evaluation-Framework-for-Kannada-LLM-Summarization
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running the Application

Run the Streamlit application:

```bash
streamlit run app.py
```

The application will launch locally in your browser.

---

## Sample Workflow

1. Enter original Kannada text
2. Provide summaries from multiple LLMs
3. Click "Evaluate Models"
4. View:
   - comparative metrics
   - model rankings
   - visualization charts
   - final weighted scores
5. Download results as CSV
